# LLMaestro
This is a UTK CS429 project to see whether chord notation affects LLM generation of chord progression.

## Steps
Download ChoCo dataset zip from https://github.com/smashub/choco/releases/tag/v1.0.0

Place the downloaded ChoCo zip inside the project directory and unzip it.

Create a Python virtual environment, activate the environment, and pip install jams
